# Project Documentation for Auckland Interior Detail

## Features
- High-quality interior detailing services for vehicles.
- Eco-friendly cleaning solutions and techniques.
- Convenient mobile service available.
- Online booking and scheduling.
- Customer loyalty program.

## File Structure
```
aucklandinteriordetail/
├── src/
│   ├── components/      # React components
│   ├── pages/           # Application pages
│   ├── styles/          # Stylesheets
│   └── utils/           # Utility functions
├── public/              # Static files
├── README.md            # Project documentation
└── package.json         # Project dependencies and scripts
```

## Deployment Information
- The application is deployed using Vercel.
- The main branch is automatically deployed to production.
- Environment variables for API keys are stored securely in Vercel.

## Development Guidelines
1. Clone the repository using `git clone <repo-url>`.
2. Install dependencies with `npm install`.
3. Run the application locally using `npm start`.
4. Follow the coding standards defined in `.eslintrc`.
5. Submit pull requests for code reviews before merging into the main branch.
6. Ensure all tests are passing before pushing to the main branch.
